Три шага — и завод работает

Вы определяете тему. Bot Zavod делает остальное.

// 01
📡

Сбор сигналов

Завод читает RSS-ленты, Telegram-каналы и сайты по вашей теме. Фильтрует шум, убирает дубли — остаётся только релевантное.

// 02
✍️

AI пишет посты

LLM переписывает сигналы в готовые посты в вашем тоне. Quality Gate проверяет качество. Вы можете одобрять вручную или доверить полностью AI.

// 03
📤

Публикация по расписанию

Посты выходят в ваш канал по заданному расписанию. Правильное время, правильный темп — автоматически.

// что внутри

Полный пайплайн

📰

RSS-источники

Любые RSS-ленты по вашей теме. Фильтрация по ключевым словам и релевантности.

✈️

Telegram-каналы

Парсинг публичных каналов и чатов по теме. Сигналы из первоисточника.

🌐

Web-мониторинг

Слежение за сайтами на изменения. Статьи, анонсы, обновления — без RSS.

🧠

AI-классификация

LLM определяет тему, тип и качество каждого сигнала. Только релевантное идёт дальше.

🔍

Дедупликация

Векторный поиск (Qdrant) находит похожие материалы. Одно и то же — не выйдет дважды.

✅

Quality Gate

Автоматическая проверка качества поста до публикации. Плохой контент отсеивается.

📅

Менеджер расписания

Таймзоны, лимиты публикаций в день, выходные. Канал живёт правильным ритмом.

📊

Мониторинг здоровья

Пайплайн следит за собой. Сбои фиксируются, ежедневный отчёт — к вам.

🖼️

Генерация картинок скоро

AI-иллюстрации к постам (DALL-E / Flux). Текст + картинка — в Фазе 1.
