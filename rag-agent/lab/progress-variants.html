Прогресс-бар: 5 вариантов

Все примеры — состояние «шаг 3 из 7». Шаги 1–2 done (галочка teal), шаг 3 current (лайм fill), шаги 4–7 pending. Карточка-обёртка одинаковая, отличается только подача цифры/маркера.

Вариант 1 Моно с точкой-разделителем

Цифра mono 11px → middle dot → лейбл. Без круга вокруг цифры. Done — галочка вместо цифры. Стиль терминала / IDE.

Вариант 2 Цифра как watermark

Цифра mono крупная (15px), но на pending — приглушена до 25% opacity (как водяной знак). На current/done — opacity 1, читается громко. Без круга, без точки.

Вариант 3 Квадратный чип вместо круга

Mini-square 22×22 с rounded-radius 5px. Контраст форм (овальный pill ↔ квадратный чип). Pending — graphite, current — белый чип на лайме, done — teal-fill с галкой.

Вариант 4 Stepper с соединительной линией

Шаги связаны линией. До текущего шага линия teal→accent градиент, после — серая. Цифры в круглых dot'ах на линии. Классический form-stepper, сильнее «двигает» взгляд по прогрессу.

Вариант 5 Таблоид: только лейбл + цветной dot

Цифр нет вообще. Маркер — крошечная точка 7px слева. Pending — серая, current — тёмная с glow на лайме, done — teal-fill с галкой. Минимум информации, максимум визуальной чистоты.
